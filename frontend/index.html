MediLink
Plateforme de gestion des ordonnances médicales
🏥
Microservices Spring Boot avec communication Feign + RabbitMQ
🔐
Sécurisé par Keycloak — gestion des rôles DOCTOR / PATIENT / ADMIN
🗄️
MySQL (ordonnances) + MongoDB (notifications) — architecture polyglotte
🔔
Notifications temps réel via RabbitMQ asynchrone
