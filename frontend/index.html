⚙️ Obtenir un Token Keycloak

Configurez le realm medilink dans Keycloak sur http://localhost:8080 puis créez les rôles : DOCTOR, PATIENT, ADMIN

📖 Guide de démarrage rapide

  1. Lancer : docker-compose up --build
  2. Attendre que tous les services soient UP (~2-3 min)
  3. Ouvrir Keycloak : http://localhost:8080 (admin/admin)
  4. Créer realm medilink avec client medilink-client
  5. Créer rôles : DOCTOR, PATIENT, ADMIN
  6. Créer un user avec rôle DOCTOR, obtenir son token ici
  7. Coller le token dans le champ JWT en haut et tester les APIs
  8. Eureka : http://localhost:8761
  9. RabbitMQ UI : http://localhost:15672 (guest/guest)
  10. Swagger Ordonnances : http://localhost:8085/swagger-ui.html
  11. Swagger Notifications : http://localhost:8084/swagger-ui.html
